# Research-Project-Data-Mining
This repository contains the final research project for the **Data Mining and Data Warehouse** course at **UNITBV**.
## Authors
* **Aimar Alustiza**
* **Fermín Barrena**

## Project Description
The main objective is not to find the "perfect" model, but rather to measure the impact of the **non-transferability of preprocessing pipelines**.

Two families of classifiers are compared:
1. **Distance Family:** K-Nearest Neighbors (KNN) and Logistic Regression.
2. **Partition Family (Trees):** Decision Tree and Random Forest.
